How to install Coq on various systems

The Coq website contains a page with instructions on how to install Coq and related packages via opam. This is the recommended method of installation as opam can then be used to install third-party Coq libraries maintained by the community. Note however that installing and using opam on Windows requires the use of a Linux compatibility layer such as WSL or MINGW.

Additionally, the Coq wiki contains three helpful pages detailing various ways of installing Coq for the three most common operating systems:

  • Linux
  • Windows
  • MacOS

Previous and development versions of Coq

The development version of Coq is browsable and downloadable from our Git repository using command: git clone https://github.com/coq/coq.git.

Most of the previous versions of Coq are available on GitHub or here.
