What is OPAM

OPAM is the package manager for the OCaml programming language, the language in which Coq is implemented. Instructions on how to install OPAM itself are available on the OPAM website.

By following the instructions on this web page one installs the last stable version of Coq and all additional packages in a switch with-coq-<#CURRENTVERSION>. Instructions target an OPAM newcomer.

Using OPAM to install Coq

Once the opam command is available, i.e. OPAM is installed, one can proceed as follows (using opam 2):


# Initialize the opam root (at ~/.opam by default) and create a switch named "with-coq-<#CURRENTVERSION>"
# This will prompt you to allow opam to set up initialization scripts, which is generally fine to accept.
# "-j 2" is the number of CPU cores to use.
# By default the global OCaml installation will be used. opam can also install and use a different
# version of OCaml (e.g., <#OCAMLV>) by adding here the option "--compiler=ocaml-base-compiler.<#OCAMLV>".
opam init --switch=with-coq-<#CURRENTVERSION> -j 2

# Activate the repository for Coq packages.
opam repo add coq-released http://coq.inria.fr/opam/released

# Pin the coq package to version <#CURRENTVERSION> and install it.
opam pin add coq <#CURRENTVERSION>

If using opam 1.2, the init command should be:

opam init -n --comp <#OCAMLV>

One may also want to install CoqIDE. Note that this requires GTK+ development files (gtksourceview2) to be available on the system.

opam install coqide

For alternative user interfaces / editors, see instructions on their own homepage, e.g. https://proofgeneral.github.io/#quick-installation-instructions for Proof-General.

Running Coq

If you couldn't allow opam to set up initialization scripts at the opam init step above, every time a new shell is opened you have to type in the following lines in order to use Coq:


eval `opam config env`

To check that the installation was successful, coqc -v should print the version of Coq.

Using OPAM to install Coq packages

If Coq has been installed as described above, the list of available Coq packages can be accessed as follows

opam search coq

The command lists the Coq package names followed by a short description. One can access a more detailed description of a package, say coq-sudoku, by typing

opam show coq-sudoku

One can install the package by typing

opam install coq-sudoku
